# 📌 Board Web Application

Spring Boot 기반의 게시판 웹 애플리케이션으로  
회원 관리, 게시글 CRUD, 파일 업로드 기능을 구현한 프로젝트입니다.

# 🛠 Tech Stack

### Backend
- Spring Boot 3
- MyBatis

### Frontend
- Thymeleaf
- Bootstrap

### Database
- Oracle XE

### Build Tool
- Gradle

### 기타
- Jsoup (XSS 방어)
- File Upload (Multipart)

---

# 📌 주요 기능

## 👤 회원 기능
- 회원가입 / 로그인 / 로그아웃
- 회원 정보 수정 / 탈퇴

---

## 📝 게시판 기능
- 게시글 작성 / 수정 / 삭제 / 조회
- 답글 기능 (계층형 구조)
- 페이징 처리

---

## 📎 파일 업로드
- 파일 업로드 및 다운로드
- DB(BLOB)에 파일 저장
- 이미지 파일 처리
